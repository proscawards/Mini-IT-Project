Long Distance Bus Ticket Booking Program(PSP0201)
---------------------------------------------------------------------
CAUTION: Please EXTRACT the ITProject.rar to Desktop.
---------------------------------------------------------------------
How to run the program:
1. Open Command Prompt(cmd)
2. Type cd Desktop
3. Type cd ITProject
4. Type py Mainfile.py
or just open Mainfile.py inside ITProject folder.
---------------------------------------------------------------------
Guidelines of using the program
---------------------------------------------------------------------
For new user:
1. Register an account
2. Login
3. Insert amount of ticket(s) 
4. Insert I/C no.
5. Select Departure Time and Date
6. Select Seat
7. Choose the option of whether to include the insurance
8. Click NEXT to continue; if click BACK, repeat from step 6
9. Click CONFIRM
10.Click anywhere to continue to exit the program.
---------------------------------------------------------------------
For returning user:
You can directly Login to view past booking(s). 
To purchase tickets, repeat from step 3 to step 10.
